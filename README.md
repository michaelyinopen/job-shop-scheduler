## How to run locally

You'll need [node](https://nodejs.org/) and [yarn](https://classic.yarnpkg.com/en/docs/install).


Install nwb with
```
npm install -g nwb@0.23.x
```

Install packages
```
yarn install
```

Start the demo project with
```
yarn start
```
